# 🚀 Projects so far:
## Anonymous Chat Platform - PopChats:
A public, anonymous, real-time chat system with no login and no persistent storage >>
- Temporary usernames with no account system
- Fully ephemeral chats (auto-deleted on server reset)
- Built by leveraging Render free-tier behavior (server sleep → memory reset)
- Zero data persistence — no logs, no storage
- Basic moderation and rate limiting to prevent spam/abuse
- Designed around system constraints to create a disposable chat environment

**Key Idea:** Instead of solving infrastructure limitations, the system uses them as a feature — creating a naturally self-destructing chat environment.

- [View Repo.](https://github.com/NILAY-KS07/PopChats)<br>
- [Live Demo<br>(*Note: The server may take up to a minute to start due to free-tier hosting.*)](https://popchats.onrender.com)
